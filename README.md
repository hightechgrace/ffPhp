ffPhp
=====
_Forms for PHP_

Overview
--------

With ffPhp you can create xHtml forms in php very easy.

Supported controls are:

 * Fieldset
 * Singleline input
 * Multiline input (textarea)
 * Hidden field
 * Checkbox group
 * Radiobutton group
 * Grouped and ungrouped lists

Look into `demo.php` for some examples.
