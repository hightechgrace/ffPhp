ffPhp
=====
_Forms for PHP_

Overview
--------

With ffPhp you can create xHtml forms in php very simple.

Supported controls are:
 * Fieldset
 * Singleline input
 * Multiline input (textarea)
 * Hidden field

 * more coming...

Look into `demo.php` for some examples.
